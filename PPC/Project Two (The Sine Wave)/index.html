The Sine Wave

What are Sine Waves?

A sine wave is a fundamental type of wave, with smooth, repeating, and symmetrical curves that oscillates regularly between maximum and minimum values. Sine waves are characterised by its frequency, which determines how quickly it repeats, and its amplitude, which indicates height.

Sine Wave Visualisation using JavaScript

let xspacing = 16; 
let w; 
let theta = 0.0; 
let amplitude = 75.0; 
let period = 500.0; 
let dx;  
let yvalues; 

function setup() {
  strokeWeight(1)
  createCanvas(windowWidth, 500);
  w = width + 16;
  dx = (TWO_PI / period) * xspacing
  yvalues = new Array(floor(w / xspacing))
}

 

function draw() {
  background(255); 
  calcWave()
  renderWave();
}


function calcWave(){
  theta += 0.02

  let x = theta;
  for(let i = 0; i < yvalues.length; i++){
    yvalues[i] = sin(x) * amplitude;
    x += dx;
  }
}

function renderWave(){
  noStroke()
  fill(0)
  for(let x = 0; x < yvalues.length; x++){
    ellipse(x * xspacing, height / 2 + yvalues[x], 16, 16)
  }
}
  1. Variable Declarations:
    • 'xspacing':
      This variable is set to 16 and determines the horizontal spacing between wave points.
    • 'w':
      It will store the width of the canvas.
    • 'theta':
      Represents an angle, initialised to 0.0, it is used to animate the wave.
    • 'amplitude':
      The amplitude of this sine wave is set to 75.0, this controls the height of the sine wave.
    • 'period':
      The period of the wave is set to 500 pixels. This variable determines how many oscillations the wave completes within the canvas.
    • 'dx':
      It's calculated as two pi divided by the period multiplied by the x spacing, and determines how much theta (the angle) chnages in each itteration.
    • 'yvalues':
      This is an array used to store the y-coordinates of the wave points.

  2. setup() Function
    • 'setup()':
      setup() is a p5.js function that is called once when the program starts.
    • 'strokeWeight(1)':
      strokeWeight() defines the weight, in this case, the stroke weight is one pixel.
    • 'createCanvas(windowWidth, 500)':
      This creates an HTML5 canvas with a width equal to the width of the window and a fixed height of 500 pixels.
    • 'w':
      The function we previouly declared 'w', is initialised to width + 16. Which is the width of the canvas plus an offset.
    • 'dx':
      The difference in the x value.
    • 'yvalues':
      An array of y values is created, with the y value being determined by the width divided by the x spacing.

  3. draw() Function
    • 'draw()':
      draw() is another p5.js fnction that continuously executes as long as the program is running.
    • 'background()':
      This sets the background color to black.
    • 'calcWave()':
      This calls the calcWave function which is defined below.
    • 'renderWave':
      This calls the renderWave function which is defined below.

  4. calcWave() Function
    • 'theta += 0.02':
      This increments theta by a value of 0.02 pixels per frame to animate the wave.
    • It then iterates through the yvalues array. For each element, it calculates the sine of x, which is initially equal to theta and multiplies it by the amplitude. The result is stored in the yvalues array.
    • 'x' is incremented by dx in each iteration, causing the wave to move horizontally.

  5. renderWave() Function
    • 'renderWave()':
      Is responsible for rendering the waves on the canvas.
    • It sets the stroke to noStroke() (no outline), and then fills the color to black.
    • It then loops through the y values array and draws ellipses at the calculated x and y coordinates to represent the wave. The wave points are evenly spaced horizontally and hae a fixed size of 16 pixels by 16 pixels.
